## Usage

Clone this repository and open `BitBay-Ticker.xcodeproj`.

```bash
$ git clone https://github.com/albinekcom/BitBay-Ticker-iOS.git
$ cd BitBay-Ticker-iOS
$ open BitBay-Ticker.xcodeproj
```
